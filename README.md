# allekirjoituspalvelu
Allekirjoituspalvelu on selkeä ja turvallisen tuntuinen PDF-dokumenttien sähköieen allekirjoittamiseen. Käyttäjä voi ladata PDF-tiedoston, lisätä allekirjoittajat, lähettää allekirjoituspyynnön ja seurata dokumentin tilaa reaaliaikaisesti. Pankkitunnistautuminen näkyy käyttöliittymässä valmiina osiona, mutta sen tekninen toteutus lisätään myöhemmin
